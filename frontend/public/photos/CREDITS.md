# Photo credits

Placeholder photography for the Hazava AI interface. All images are from
[Unsplash](https://unsplash.com) under the [Unsplash License](https://unsplash.com/license)
(free to use, no attribution required — credit included here as good practice).

They are rendered with an azure duotone treatment in the UI (`<Duotone>`), so the
underlying photo can be swapped for real Doctors for Madagascar field photography by
replacing the file of the same name — no code change needed.

| `care.jpg` | Report / section header band (optional) | unsplash.com/photos/1584515933487-779824d29309 |

**To use real DFM photos:** drop a replacement JPG with the same filename into this folder.
Recommended subjects: clinic/maternal-child care, field health workers, community, and
Madagascar landscapes. Aim for ~1600–1920px wide for the hero, ~900px for the smaller slots.
